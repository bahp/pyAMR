Contents of this folder
-----------------------
URL: http://antimicrob.net/wp-content/uploads/DRI-instructions_engl.pdf

summary.csv
    The sample table provided by CDDEP (URL) to compute the DRI.

outcome.csv
    The expected output as computed by CDDEP.

susceptibility.csv
    The summary table part related to susceptibility test records formatted as
    if they would have been extracted from the database. It contains columns
    such as DATE, MICROORGANISM, ANTIMICROBIAL, SENSITIVITY.

    .. note:: Generated by us.

prescriptions.csv
    The summary table part related to prescription records formatted as
    if they would have been extracted from the database. It contains columns
    such as DATE, MICROORGANISM, DRUG, DOSE

    .. note:: Generated by us.
