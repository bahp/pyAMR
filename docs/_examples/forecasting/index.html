Time-series forecasting¶

The examples presented in the gallery below show how to use some of the tools provided in this module to forecast trend. The forecast methods supported are:

  • WLS: Weighted Linear Squares

  • ARIMA: Auto Regressive Integrated Moving Average

  • SARIMAX: Seasonal Auto Regressive Integrated Moving Average with eXogenous regressors
